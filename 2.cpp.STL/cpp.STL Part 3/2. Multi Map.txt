//2.Multi Map 
A multi-map is a container in the C++ Standard Template Library (STL) 
that allows storing key-value pairs in which keys are sorted in a specific order, 
and multiple elements with the same key are allowed.

//Syntax 
multimap <Key_type,Value_type> map_name;

//Properties
Sorted MultiMap is sorted based on the keys, typically in ascending order.

Not Unique It allows multiple elements with the same key.

//Types of Declarations
You can declare a map with various data types as keys and values. Common examples include:

multimap<int, int>: Map with integer keys and integer values.

multimap<int, pair<int, int>>: Map with integer keys and a pair of integers as values.

multimap<pair<int, int>, int>: Map with a pair of integers as keys and integer values.

multimap<char, string>: Map with character keys and string values.


//example 

#include<bits/stdc++.h>
using namespace std;

int main()
{
    // Multimap declaration
    multimap<int, int> mm1;
    multimap<int, pair<int, int>> mm2;
    multimap<pair<int, int>, int> mm3;
    multimap<char, string> mm4;

    // Inserting elements
    mm1.insert({1, 3});
    mm1.insert(make_pair(7, 5));
    mm1.emplace(4, 8);
    mm1.emplace(4, 9);  // Multimap allows multiple keys

    // Iterating over multimap elements
    for(auto& kv : mm1) {
        cout << kv.first << " " << kv.second << endl;
    }

    // Finding all values for a particular key
    cout << "All values for key 4:" << endl;
    auto range = mm1.equal_range(4);
    for(auto it = range.first; it != range.second; ++it) {
        cout << it->second << endl;  // Should print both 8 and 9
    }

    // Finding a single element
    auto iter = mm1.find(7);
    if (iter != mm1.end()) {
        cout << "Value for key 7: " << iter->second << endl;  // Should print 5
    }

    // Trying to find a key that doesn't exist
    auto iter_1 = mm1.find(10);
    if (iter_1 == mm1.end()) {
        cout << "Key 10 not found." << endl;
    }

    // Using multimap with different data types
    mm4.insert({'a', "apple"});
    mm4.insert({'b', "banana"});
    mm4.emplace('a', "avocado");  // Duplicates allowed for key

    for(auto& kv : mm4) {
        cout << kv.first << " " << kv.second << endl;
    }

    return 0;
}

//explanation 

//multimap<k, v> mm: 
Declares a multimap named mm that stores key-value pairs with keys of type k and values of type v.

//mm.insert({k, v}): 

Inserts the key-value pair {k, v} into the multimap mm.

This method always adds a new element to the container since multimap allows duplicate keys.

//mm.emplace(k, v):

Similar to insert, this function attempts to construct the key-value pair in-place within the multimap, adding the pair (k, v) to the container.

//mm.find(k):
Searches the multimap mm for the first occurrence of key k and returns an iterator to it.

If the key is not found, it returns an iterator to multimap::end().

Since multimap can contain multiple elements with the same key, this only provides access to one such element (typically the first encountered in the underlying structure).

//mm.equal_range(k): 
Returns a pair of iterators that represent the range of elements with the key k.

The first iterator points to the first element of the range, and the second iterator points to one past the last element of the range.

//(*iter).second: 

Using the iterator iter, accesses the value part of the key-value pair.

//for(auto& kv : mm): 
This range-based for loop iterates over all key-value pairs in the multimap mm.
kv is a reference to each pair, where kv.first is the key and kv.second is the value.

//mm.end(): 
This is used to check whether a search was successful by comparing the result of find or other search operations to mm.end().
